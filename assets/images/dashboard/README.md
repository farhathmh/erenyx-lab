# Dashboard Image Naming

Store dashboard captures in this folder using clear, stable names.

## Standard Names

- erenyx-dashboard-homepage.png
- erenyx-dashboard-services.png

## Current Scope

- Desktop dashboard captures only.
- Mobile dashboard capture deferred for a future update.

## Capture Rules

- Prefer PNG format for readability.
- Use full-width desktop capture for homepage.
- Re-capture after major dashboard layout changes.
- Keep sensitive hostnames or secrets hidden.
