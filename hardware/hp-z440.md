# HP Z440 Workstation — Proxmox Host

## Specifications

| Component | Detail |
|-----------|--------|
| CPU | Intel Xeon E5-2698 v4 — 20 cores / 40 threads @ 2.2GHz base |
| RAM | 64GB DDR4 ECC Registered |
| Storage 1 | 960GB SSD — Proxmox OS + VM storage |
| Storage 2 | 1.5TB HDD — ISO images, VM templates, backups |
| Storage 3 | 1TB HDD — PCIe passthrough to TrueNAS CE |
| Form factor | Tower workstation |
| Acquired | Second-hand market |

---

## Role in the Lab

Primary hypervisor running Proxmox VE bare metal. This machine handles
all virtualisation workloads including network simulation and storage.

---

## Running VMs

| VM | OS | Purpose |
|----|-----|---------|
| EVE-NG CE | Ubuntu base | Network simulation — CCNA and CCNP lab topology practice |
| TrueNAS CE | TrueNAS SCALE | Network attached storage — 1TB HDD passed through via PCIe |

---

## Why This Hardware

The Xeon E5-2698 v4 provides 40 threads which is critical for running
multiple VMs simultaneously without performance degradation. ECC RAM
ensures memory error correction — important for a storage VM like TrueNAS.
The PCIe passthrough capability allows direct hardware access to the HDD
from TrueNAS for proper drive health monitoring and ZFS functionality.
